Important Concepts

Lesson 1: The Basics of the Web and HTML

The Web
a collection of HTML (i.e. text) documents
Major pieces:
browsers, servers, protocol: HTTP
HTML
consists of:
  • text
  • content (what you see),
  • markup (what it looks like),
  • references to documents (e.g. videos),
  • links to other pages
Markup:
made up of tags: <a>= opening tag and </a>closing tag
An Element is the sequence: tag/content/tag
Examples for and elements of tags
  • <b>=bold
  • <em>=emphasis/Italic
  • Attributes: property of HTML tag; is a sequence: opening tag ATTR= value content closing tag
  • <a>: anchor tag for links (attr: href)
  • <img>=image, attributes: src= url alt= text (alt: when image does not load, required!)
  • void tag: no closing tag
  • Whitespace:<br> break: text on multiple lines; line break (void tag)
  • paragraph tag:<p> needs closing tag
inline element vs. block element:
creates an invisible box: e.g. p or span (inline) vs. div (block)
Structure of HTML document
doctype, hmtl, head (CSS, Java Script), body

and: Computers are stupid: Be exact!
